# Черновой DCD-схемы 

> Усреднённый черновой вариант — три ОДУ ( $dL/dt$, $dC/dt$, $dS/dt$ ), переменные, параметры и привязка к уровням NIMAC - усреднены. 

---

Ниже — прагматичная, фальсифицируемая черновая формализация Differentiable Consciousness Dynamics (DCD). Я даю уравнения в «полузвёздном» виде (основные член/термы), объясняю смысл каждого параметра, указываю привязку к уровням NIMAC (L0–L13) и предлагаю измеримые прокси + экспериментальные вмешательства для идентификации параметров. Эта схема ориентирована на быструю реализацию в моделировании и на планирование мульти-уровневых экспериментов. 

---

## 1. Переменные состояния (state variables)

* **$L(t)$** — *Level*: уровень глобальной интеграции / arousal / доступности (шкала 0…10). (Связан с GNW, PCI и интеграцией). 
* **$C(t)$** — *Content*: объём / дифференцированность содержимого (размер смыслового manifold, насыщенность репрезентаций) (0…10). (Связан с локальными популяционными кодами, P3b, MVPA). 
* **$S(t)$** — *Self / Metacognition*: состояние мета-представлений / интероцептивного фона / уверенности (0…10). (Связан с DMN/aPFC, interoception). 

Каждая переменная нормирована в удобной шкале (0–10) для сравнимости с координатами из Фреймворк «Трёх фундаментальных измерений сознания». 

---

## 2. Базовая форма уравнений (полу-реалистичная, с нелинейностями)

$$
\boxed{\begin{aligned}
\tau_L ,\frac{dL}{dt} &= -\big(L - L_0\big) + G_L(L);+;w_{LC},G_C(C);+;w_{LS},G_S(S)\\
&\quad +;P_{NV}(t);+;I_L(t);+;\eta_L(t)\\[6pt]
\tau_C ,\frac{dC}{dt} &= -\big(C - C_0\big) + w_{CL},H_L(L);-;w_{CD},D(C)\\
&\quad +;w_{CS},H_S(S);+;I_C(t);+;\eta_C(t)\\[6pt]
\tau_S ,\frac{dS}{dt} &= -\big(S - S_0\big) + w_{SL},U_L(L);+;w_{SC},U_C(C) \\
&\quad -;A_S(S);+;I_S(t);+;\eta_S(t)
\end{aligned}}
$$

Где:

* ( $\tau_{L,C,S}$ ) — характерные временные константы (в секундах) для каждой оси.
* ( $L_0, C_0, S_0$ ) — «базовые»/гомеостатические значения (доминирующие точки при отсутствии входов).
* ( G_*,H_*,U_* ) — нелинейные трансферные функции (например sigmoid / gain functions или Hill-type).
* ( $D(C)$ ) — «детерминант» / насыщение содержимого (контролирует перегрузку информацией / дизбиндинг).
* ( $A_S(S)$ ) — адаптация/утомление метакогнитивной системы (homeostatic slow fatigue).
* ( $P_{NV}(t)$ ) — нейроваскулярный/метаболический вклад (поддержка L через L5 / L11).
* ( $I_{L,C,S}(t)$ ) — внешние входы/манипуляции (сенсорика, TMS, фармакология, стимулы).
* ( $\eta_{*}(t)$ ) — стохастические шумовые члены (коррелированные/бело-шумовые).

---

## 3. Предложенные конкретные формы трансфер-функций (для моделирования)

(используются стандартные формы: сигмоида, кратная насыщение)

* ( $G_L(L) = g_L\cdot\mathrm{sigm}(\alpha_L (L - \theta_L))$ ) — самоподдерживающий feedback GNW-типа (ignition).
* ( $G_C(C) = g_C\cdot\mathrm{sigm}(\alpha_C (C - \theta_C))$ ) — «разгоревание» содержимого, усиливающее L через broadcast.
* ( $H_L(L) = h_L\cdot L/(1 + \kappa_L L)$ ) — gain of content by level (упрощённый saturating gain).
* ( $D(C)=d_C\cdot C^2$ ) — квадратичная перегрузка (высокое C может снижать стабильность).
* ( $U_L(L)=u_L\cdot\mathrm{sigm}(\alpha_{SL}(L - \theta_{SL}))$ ) — L → S (направляющее влияние уровня на метакогницию).
* ( $A_S(S)=a_S\cdot S/(1+\rho_S S)$ ) — медленная адаптация/утомление S.

Эти формы гибки и заменяемы; главное — включить нелинейные пороги и насыщения, чтобы модель могла порождать аттракторы и флипускции (bistability, ignition). 

---

## 4. Привязка параметров и членов уравнений к уровням NIMAC

Ниже — таблица «терм → NIMAC уровни» (ключевая для экспериментов).

* **Самоподдерживающий член ( $G_L(L)$ ) (ignition)** → **L1 (GNW)**, опосредован глобальной интеграцией; зависит от межсетевой / крупномасштабной связности (L2–L3) и от колонковой организации (L4). 
* **$w_{LC}$ (C → L gain)** → локальные представления (L4–L6) и их broadcast через GNW (L1). Изменяется нейромодуляторами (L11). 
* **$P_{NV}(t)$** (нейроваскулярная поддержка, ресурсный вклад) → **L5 (neurovascular unit)** и **L11 (метаболика)**: меняет ( $\tau_L$ ) или амплитуду $G_L$. 
* **$D(C)$ (перегрузка)** → отражает синаптическую/клеточную исчерпаемость: **L6–L8** (клеточные, субклеточные, молекулярные механизмы), влияет на скорость восстановления и устойчивость содержимого. 
* **S-члены ( $w_{SL}$, $w_{SC}$ )** → **L2 (DMN / межсетевые self-networks)**, **L6 (PFC ensembles)** и **L11 (нейрохимия интероцепции)**; S возникает из сочетания высокого-уровневой интеграции и интероцептивного фона. 
* **Эфаптические / полевые влияния** → **L12**: могут модифицировать синхронизацию и effective coupling, добавляя быстрые (мс–с) кросс-члены между L и C. 
* **Спекулятивные квантовые/микротрубочные факторы** → **L13**: в модели включаются как малая модуляция шума/коерентности (фактор, влияющий на коррелированный шум ( $\eta$ )), но оставляются по возможности экспериментально нейтральными до верификации. 

(Кратко: L1–L4 обеспечивают структуру $G_L$ и межсетевую передачу; L5/L11 обеспечивают ресурсы и временные константы; L6–L8 реализуют локальные ограничения и пластичность; L12/L13 — дополнительные корреляционные модификаторы). 

---

## 5. Рекомендованные значения/диапазоны временных констант и интерпретация (ориентиры)

(значения — ориентировочные; уточняются под конкретные экспериментальные постановки)

* ( $\tau_L$ ) — 0.1 … 2 s (GNW-timescale; ignition ~100–1000 ms, но влияние метаболики может растянуть эффект). 
* ( $\tau_C$ ) — 0.01 … 0.5 s (ERP/спайк-популяции; быстрый манипулируемый контент). 
* ( $\tau_S$ ) — 1 … 30 s (метакогниция и interoceptive integration часто медленнее, секунды–десятки секунд). 

Параметры веса ( $w_{ij}$ ) обычно в диапазоне 0 … 5 (безразмерные коэффициенты усиления) — в симуляциях их подбирают методом чувствительного анализа. Нелинейные пороги ( $\theta_*$ ) устанавливают критические точки ignition / collapse.

---

## 6. Проксии для параметров — что измерять в экспериментах

(чтобы оценить параметры, какие данные собирать)

* **( $\tau_L, G_L$ )**: TMS-EEG → PCI dynamics; восстановление PCI после perturbation; EEG spectral exponent dynamics (L1/L2). 
* **$w_{LC}$, $w_{CL}$**: cross-modal decoding (fMRI MVPA), TMS→decoding gain, PLV/phase-locking between regions (L3–L4). 
* **$P_{NV}(t)$, $\tau_{NV}$**: simultaneous fMRI (BOLD) + MRS (metabolites) or NIRS; measure coupling of neural events to hemodynamics (L5, L11). 
* **S-параметры**: meta-d′ tasks (confidence), heartbeat-evoked potentials, DMN connectivity (fMRI), rTMS to aPFC and consequent change in meta-performance. 
* **$D(C)$**: measure performance degradation under high-information load, neural adaptation, synaptic depression proxies (paired-pulse TMS, iEEG spike suppression). 

---

## 7. Идентификация параметров и экспериментальный workflow (практика)

1. **Start small — perturbative identification**: оценить сначала ( $\tau_L, G_L$ ) с TMS-EEG (встроенная, сильная манипуляция L1). Затем зафиксировать их и оценить ( $w_{LC}, w_{CL}$ ) с задачами декодирования содержимого (fMRI/MVPA + EEG). 
2. **Use closed-loop perturbations**: real-time decoding → таргетированная стимуляция (закрытый цикл) для выявления causal w-параметров (Фреймворк «Трёх фундаментальных измерений сознания» предлагает подобную практику). 
3. **Multi-timescale recordings**: совместные записи (EEG/iEEG + fMRI + MRS) чтобы связать fast C/L dynamics с медленной метаболикой (L5/L11). 
4. **Parametric pharmacology**: модуляторы (NE, ACh, orexin) — изменяют gain и ( $\tau$ ), полезно для ограничения ( $g_L, g_C$ ) и mapping на L11. 
5. **Estimation methods**: state-space modelling (extended Kalman filter / particle filter), variational Bayes for nonlinear state-space, sequential Monte Carlo. Выполнить global sensitivity analysis и profile likelihood для оценки структурной идентифицируемости.

---

## 8. Примеры эксперимент-→-параметр (быстрая «шпаргалка»)

* **TMS pulse to PFC** → кратковременный поднятый ( $I_L(t)$ ): используйте ответ PCI(t) для оценки ( $G_L$ ) и ( $\tau_L$ ). (NIMAC: L1, L2). 
* **Propofol bolus** → уменьшение ( $w_{CL}$ ) и снижение ( $g_L$ ): наблюдаемые снижение PCI, снижение manifold dimensionality (C). Оценить зависимость для dL/dt и dC/dt. (Фреймворк «Трёх фундаментальных измерений сознания» обсуждает anesthetic effects). 
* **Psilocybin/psychedelics** → повышение C (widening manifold), уменьшение S (DMN suppression): model should show increase in C term and reduction in $w_{SL}$. (Фреймворк «Трёх фундаментальных измерений сознания» examples). 

---

## 9. Ожидаемые динамические феномены (что модель может породить)

* **Ignition / bistability**: при ( $G_L$ ) выше порога — переход к высокому L (rapid broadcast).
* **Dissociations L↑ / S↓ (psychedelics)**: механизм — повышение ( $g_C$ ) и снижение $w_{SL}$, приводит к состояниям с богатыми содержимыми и уменьшённой мета-интеграцией. 
* **Провалы из-за метаболики**: медленное снижение $P_{NV}$ → снижение τ_L → collapse L (DOC-like). (NIMAC подчёркивает L5/L11). 

---

## 10. Ограничения схемы и дальнейшие шаги

* **Упрощение уровней**: мы проецируем многослойную NIMAC-архитектуру на три агрегированных переменные; это удобно, но теряет детальность L7–L10 при неправильной калибровке. 
* **Идентифицируемость**: без интервенций и мульти-модальных данных многие w-параметры будут неидентифицируемы — поэтому требуется дизайн perturb + measure. 
* **L13 (квантовое)** — оставлен как опциональный фактор в шуме/корреляциях; не вводить в основные объяснения до верификации.

## Python simulation for the DCD ODEs (visible outputs)

Минимальную симуляция DCD (три ОДУ) - показал, как изменения коэффициентов связей (w_{ij}) влияют на траектории L, C, S.

Что было сделано (коротко и по делу)

* Реализована численная модель по схеме, обсуждённой ранее (Euler-интегратор, нелинейные transfer-функции: сигмоида, saturating gain, квадратичная перегрузка и адаптация для S).
* Смоделированы три сценария: baseline, сильный (w_{LC}) (усиленный вклад C → L), и слабый (w_{SL}) (ослабленный вклад L → S).
* Сохранены три отдельные фигуры (по одному графику на сценарий) и выведена таблица со статистиками (mean, max для L/C/S).

Ключевые наблюдения из симуляции

* Увеличение (w_{LC}) (C → L) заметно повышает средний и максимальный уровень L (большее global ignition / broadcast) — см. сценарий **high_w_LC**. Это демонстрирует, что усиление влияния содержимого на уровень интеграции делает систему чаще и дольше находящейся в высокоинтегрированном состоянии.
* Уменьшение (w_{SL}) (L → S) снижает среднее значение S и его вариативность (сценарий **low_w_SL**) — то есть ослабленная передача уровня в метакогнитивную систему уменьшает «самость»/метапредставления, при прочих равных L и C изменяются слабее.
* В baseline модель демонстрирует ignition-подобное поведение после сенсорных импульсов (пульсы в C), с последующим устойчивым повышением L, пока метаболическая поддержка и обратные связи не приводят систему в стационарное состояние.

```
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import os

# Simulation function implementing Euler integration
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def simulate(params, T=30.0, dt=0.01, seed=0):
    np.random.seed(seed)
    steps = int(T / dt)
    t = np.linspace(0, T, steps)
    # state arrays
    L = np.zeros(steps)
    C = np.zeros(steps)
    S = np.zeros(steps)
    # initial states (near baseline)
    L[0] = params['L0']
    C[0] = params['C0']
    S[0] = params['S0']
    
    # unpack params for speed
    tau_L = params['tau_L']; tau_C = params['tau_C']; tau_S = params['tau_S']
    g_L = params['g_L']; alpha_L = params['alpha_L']; theta_L = params['theta_L']
    g_C = params['g_C']; alpha_C = params['alpha_C']; theta_C = params['theta_C']
    h_L = params['h_L']; kappa_L = params['kappa_L']
    d_C = params['d_C']
    u_L = params['u_L']; alpha_SL = params['alpha_SL']; theta_SL = params['theta_SL']
    a_S = params['a_S']; rho_S = params['rho_S']
    w_LC = params['w_LC']; w_LS = params['w_LS']
    w_CL = params['w_CL']; w_CS = params['w_CS']
    w_SL = params['w_SL']; w_SC = params['w_SC']
    P_NV = params['P_NV']
    noise_scale = params['noise_scale']
    
    # External input schedule: brief sensory pulse to C at t=2s and t=10s
    I_L = np.zeros(steps)
    I_C = np.zeros(steps)
    I_S = np.zeros(steps)
    # pulse shape
    def pulse(center, width=0.2):
        return np.exp(-0.5 * ((t - center) / width)**2)
    I_C += 4.0 * pulse(2.0, 0.15)  # sensory input early
    I_C += 3.0 * pulse(10.0, 0.20) # sensory input later
    # small tonic drive to L
    I_L += 0.2 * np.ones_like(t)
    
    for i in range(steps - 1):
        # compute nonlinear functions
        G_L = g_L * sigmoid(alpha_L * (L[i] - theta_L))
        G_C = g_C * sigmoid(alpha_C * (C[i] - theta_C))
        H_L = h_L * L[i] / (1 + kappa_L * L[i])
        D_C = d_C * (C[i]**2)
        U_L = u_L * sigmoid(alpha_SL * (L[i] - theta_SL))
        G_S = 0.8 * sigmoid(6.0 * (S[i] - 2.0))  # small S->L influence
        
        A_S = a_S * S[i] / (1 + rho_S * S[i])
        
        # noise terms
        eta_L = noise_scale * np.random.randn()
        eta_C = noise_scale * np.random.randn()
        eta_S = noise_scale * np.random.randn()
        
        # derivatives (Euler)
        dL = ( - (L[i] - params['L0']) + G_L + w_LC * G_C + w_LS * G_S + P_NV + I_L[i] + eta_L ) / tau_L
        dC = ( - (C[i] - params['C0']) + w_CL * H_L - params['w_CD'] * D_C + params['w_CS'] * (0.5 * S[i]) + I_C[i] + eta_C ) / tau_C
        dS = ( - (S[i] - params['S0']) + w_SL * U_L + w_SC * (0.4 * C[i]) - A_S + I_S[i] + eta_S ) / tau_S
        
        L[i+1] = L[i] + dt * dL
        C[i+1] = C[i] + dt * dC
        S[i+1] = S[i] + dt * dS
        
        # keep variables within bounds (0..10)
        L[i+1] = np.clip(L[i+1], 0, 10)
        C[i+1] = np.clip(C[i+1], 0, 10)
        S[i+1] = np.clip(S[i+1], 0, 10)
    
    df = pd.DataFrame({'t': t, 'L': L, 'C': C, 'S': S})
    return df

# baseline parameters
base_params = {
    'tau_L': 0.3, 'tau_C': 0.05, 'tau_S': 5.0,
    'L0': 1.0, 'C0': 0.5, 'S0': 2.0,
    'g_L': 3.0, 'alpha_L': 8.0, 'theta_L': 3.5,
    'g_C': 2.5, 'alpha_C': 6.0, 'theta_C': 2.0,
    'h_L': 0.8, 'kappa_L': 0.2,
    'd_C': 0.12, 'w_CD': 0.9,
    'u_L': 1.2, 'alpha_SL': 4.0, 'theta_SL': 2.5,
    'a_S': 0.2, 'rho_S': 0.5,
    'w_LC': 1.5, 'w_LS': 0.5,
    'w_CL': 0.7, 'w_CS': 0.15,
    'w_SL': 0.8, 'w_SC': 0.3,
    'P_NV': 0.1,
    'noise_scale': 0.05
}

# Scenario variants
scenarios = {
    'baseline': base_params,
    'high_w_LC': {**base_params, 'w_LC': 3.0},    # stronger C -> L gain (more broadcasting)
    'low_w_SL': {**base_params, 'w_SL': 0.2}       # weaker L -> S coupling (reduced metacog influence)
}

# simulate each scenario
results = {}
for name, p in scenarios.items():
    df = simulate(p, T=30.0, dt=0.01, seed=42)
    results[name] = df

# create output directory
outdir = '/brain-networks/simulation/templates'
os.makedirs(outdir, exist_ok=True)

# Plotting: one separate figure per scenario (each figure shows L, C, S)
fig_files = []
for name, df in results.items():
    fig, ax = plt.subplots(figsize=(8,4.5))
    ax.plot(df['t'], df['L'], label='L (Level)')
    ax.plot(df['t'], df['C'], label='C (Content)')
    ax.plot(df['t'], df['S'], label='S (Self)')
    ax.set_xlabel('Time (s)')
    ax.set_ylabel('State (0-10)')
    ax.set_title(f'DCD simulation — {name}')
    ax.legend()
    ax.grid(True, linestyle=':', linewidth=0.5)
    fname = os.path.join(outdir, f'dcd_{name}.png')
    fig.savefig(fname, dpi=150, bbox_inches='tight')
    fig_files.append(fname)
    plt.close(fig)

# Summarize key statistics for each scenario
summary = []
for name, df in results.items():
    summary.append({
        'scenario': name,
        'mean_L': float(df['L'].mean()),
        'mean_C': float(df['C'].mean()),
        'mean_S': float(df['S'].mean()),
        'max_L': float(df['L'].max()),
        'max_C': float(df['C'].max()),
        'max_S': float(df['S'].max())
    })
summary_df = pd.DataFrame(summary)

# display summary table to the user
import caas_jupyter_tools as cjt
cjt.display_dataframe_to_user("DCD simulation summary", summary_df)

# Print file paths for download
for f in fig_files:
    print("Saved figure:", f)

# Return objects for further inspection if needed
results, summary_df, fig_files
```

## Интерпретация результатов

Ниже — сжатая, но детальная интерпретация результатов минимальной симуляции DCD (три ОДУ), с точными количественными наблюдениями, механизмными объяснениями в терминах NIMAC и практическими предсказаниями для экспериментов.

---

### Короткое резюме

Модель демонстрирует: 
- 1) усиление влияния содержимого на уровень интеграции (увеличение (w_{LC})) приводит к более частому и устойчивому «ignition» — высокому L; 
- 2) ослабление передачи уровня в метакогницию (понижение (w_{SL})) снижает показатель S без существенной деградации L; 
- 3) кратковременные сенсорные импульсы порождают быстрые всплески C и последующее расширение L, причём величина и устойчивость расширения зависят от связей между уровнями. 

Эти эффекты количественно подтверждаются статистикой симуляций.

---

### Ключевые числовые результаты (из симуляции)

* **Baseline** — mean_L ≈ **5.56**, mean_C ≈ **1.86**, mean_S ≈ **2.74**; max_L ≈ **8.21**, max_C ≈ **4.01**, max_S ≈ **2.94**.
* **High wₗc (w_{LC}=3.0)** — mean_L ≈ **8.68** (резкое увеличение по сравнению с baseline), mean_C ≈ **2.07**, mean_S ≈ **2.76**; max_L достигал **10.0** (ограничение по отсечке).
* **Low wₛl (w_{SL}=0.2)** — mean_L ≈ **5.26** (падение по сравнению с high_w_LC, близко к baseline), mean_C ≈ **1.81**, mean_S ≈ **2.19** (заметное снижение S).

Эти значения отражают многосекундные траектории (T=30 s) с двумя сенсорными пульсами (t≈2 s и t≈10 s) заданной амплитуды.

---

### Механистическая интерпретация (как и почему)

1. **Почему растёт L при увеличении wₗc:**
   (w_{LC}) усиливает вклад активности содержимого (C) в самоподдерживающий GNW-приёмник (G_L). Повышенный поток C→L снижает порог для ignition и поддерживает длительную глобальную интеграцию → наблюдаемая высокая средняя и максимальная L. (NIMAC: это соответствует усиленной передаче от локальных колонок / популяций — L4/L3 — к GNW (L1) и увеличению влияния нейромодуляторов L11).

2. **Почему падает S при уменьшении wₛl:**
   (w_{SL}) кодирует путевые влияния L→S (уровень интеграции формирует состояние метапредставлений). Ослабление этой связи уменьшает drive на системы самости (DMN/aPFC — L2/L6), потому S снижается при относительно неизменном L/C. Это объясняет феномен «богатого содержания при ослабленной самости» (аналогичен описанию некоторых психоделических состояний, где C↑, S↓).

3. **Роль перегрузки D(C):**
   квадратичный член D(C) ограничивает рост C при сильных стимулах или при длённой высокой активности, что предотвращает бесконтрольный рост C и вводит возможность «перегрузки» содержимого, сказывающуюся на стабильности L. При очень сильном w_{LC} есть риск того, что сочетание высокого C и D(C) породит более сложную (иногда нестабильную) динамику.

4. **Влияние метаболики / P_NV:**
   малые значения P_{NV} ограничивают долговременную способность поддерживать высокий L — моделируемая «усталость» или снижение энергетической поддержки быстро снижает τ_L/амплитуду G_L, что может приводить к коллапсу L (моделирует DOC-переходы при сниженной гемодинамической поддержке). (NIMAC: L5/L11).

---

### Конкретные предсказания / проверяемые гипотезы (как тестировать в эксперименте)

1. **Манипуляция w_{LC} (повышение C→L):**

   * Как: усилить локальную репрезентацию (ярче/сложнее стимулы), либо фармакологически повысить gain (ACh/NE).
   * Ожидается: рост PCI, увеличение длительности и частоты состояний с высоким глобальным интегрированием (L), увеличение фазовой синхронизации между крупными сетями.
   * NIMAC mapping: L4→L1 усиление; вкратце — наблюдаемые эффекты должны коррелировать с изменениями в GNW-метриках (TMS-EEG, PCI), и MVPA-различимостью контента (C).

2. **Манипуляция w_{SL} (ослабление L→S):**

   * Как: временно подавить aPFC/insula (rTMS), или фармакологически подавить метакогнитивные сигналы; верифицировать с задачами уверенности (meta-d′).
   * Ожидается: понижение S (снижение показателей метакогниции: lower meta-d′, более низкая корреляция между уверенностью и accuracy), при относительно сохранённом L.
   * NIMAC mapping: снижение эффективности L1→L2 pathway.

3. **Эффект интактной метаболики:**

   * Как: манипуляции, влияющие на нейроваскулярное обеспечение (физиологические состояния, гипогликемия, лекарства) и измерения MRS/BOLD.
   * Ожидается: снижение P_NV приведёт к постепенному падению τ_L и переходу к состоянию с низким L (как при анестезии/DOC). (NIMAC: L5/L11 критичны для поддержания L).

4. **Диссоциации L vs C vs S (предсказание):**

   * Существует набор параметрических конфигураций (напр., высокое w_{LC} + низкий w_{SL}) где C и L высоки, но S низок — это предсказание о возможных нейрофизиологических состояниях с богатыми феноменами опыта, но пониженной метакогнитивной интеграцией.

---

### Практические рекомендации для исследований (на основе наблюдений)

1. **Фокус на измерениях для верификации w-параметров:** TMS-EEG (оценка G_L и τ_L), MVPA/fMRI (оценка C), мета-задания и heartbeat-evoked potentials (оценка S).
2. **Запланировать параметрический sweep по w_{LC} и w_{SL}:** экспериментально либо через varying stimulus strength/complexity, либо фармакологически; снять мульти-модальные сигналы и оценить PCI, manifold dimensionality, meta-d′.
3. **Сопоставить динамики с NIMAC-уровнями:** при эксплантациях/моделях — привязывать наблюдаемые изменения τ и gain к L5/L11 (метаболика), L1–L4 (broadcast/binding), L2/L6 (self networks).

---

### Ограничения модели и аккуратность интерпретации

1. **Упрощённость:** агрегирование сложной многоуровневой NIMAC-структуры в три скалярные переменные теряет детализацию (внутриклинические и субклеточные механизмы сведены в параметры).
2. **Параметры не эмпирически калиброваны:** абсолютные числовые значения зависят от выбранных коэффициентов; выводы качественные/направленческие.
3. **Численная аппроксимация:** Euler-интегратор и дискретная отсечка (0..10) подходят для демонстрации, но для точного анализа стабильности/байстабилити лучше использовать более точные схемы (RK4) и выполнить чувствительный анализ.
4. **Идентифицируемость:** без целенаправленных perturb+measure многие w-параметры не будут однозначно идентифицируемы; требуются каузальные вмешательства.

---

### Конкретные следующие шаги (операционные)

1. Выполнить **чувствительный анализ** по (w_{LC}, w_{CL}, w_{SL}, w_{SC}, g_L, g_C) (параметрический sweep) и отобразить карту средних/стационарных значений L/C/S.
2. Пересчитать симуляции с RK4 и многократными сидами шума, вычислить статистику латентностей ignition (время от пульса до достижения порога L).
3. Смоделировать экспериментальные манипуляции (TMS-pulse → изменение I_L; пропофол → снижение g_L; псилоцибин → повышение g_C) и вывести конкретные, количественные предсказания для PCI, MVPA, meta-d′.
4. При наличии мульти-модальных данных — провести параметрическую подгонку (state-space estimation) и протестировать идентифицируемость параметров.
