- **IIT** (Oizumi et al., 2014) –ø—Ä–µ–¥—Å–∫–∞–∑—ã–≤–∞–µ—Ç, —á—Ç–æ Œ¶ –≤–æ–∑—Ä–∞—Å—Ç–∞–µ—Ç —Å —É–≤–µ–ª–∏—á–µ–Ω–∏–µ–º –∫–∞–∫ **–∏–Ω—Ç–µ–≥—Ä–∞—Ü–∏–∏** (L), —Ç–∞–∫ –∏ **–¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏–∞—Ü–∏–∏** (C)
- **GNW** (Dehaene et al., 2017): –±–æ–≥–∞—Ç–æ–µ —Å–æ–¥–µ—Ä–∂–∞–Ω–∏–µ (–≤—ã—Å–æ–∫–∏–π C) —É—Å–∏–ª–∏–≤–∞–µ—Ç —Ä–µ–∫—É—Ä—Ä–µ–Ω—Ç–Ω—ã–µ —Å–≤—è–∑–∏ ‚Üí –ø–æ–≤—ã—à–∞–µ—Ç L —á–µ—Ä–µ–∑ sustained activation

**–≠–º–ø–∏—Ä–∏–∫–∞:**
- **Siclari et al. (2017)**: –≤ REM-—Å–Ω–µ (–≤—ã—Å–æ–∫–∏–π C) –Ω–∞–±–ª—é–¥–∞–µ—Ç—Å—è –ø–æ–≤—ã—à–µ–Ω–Ω–∞—è PCI (L) –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å N3 (–Ω–∏–∑–∫–∏–π C, –Ω–∏–∑–∫–∏–π L)

#### 2.2.3 –¢—Ä–µ—Ç–∏–π —á–ª–µ–Ω: –°—Ç–æ—Ö–∞—Å—Ç–∏—á–µ—Å–∫–∏–π —à—É–º
```

```

- **Œµ_L(t)** ‚Äî –±–µ–ª—ã–π –≥–∞—É—Å—Å–æ–≤ —à—É–º —Å ùîº[Œµ_L] = 0, Var[Œµ_L] = 1
- **Œ≥_L** = 0.1 (–∞–º–ø–ª–∏—Ç—É–¥–∞ —Ñ–ª—É–∫—Ç—É–∞—Ü–∏–π; –æ—Ç—Ä–∞–∂–∞–µ—Ç —Å–ø–æ–Ω—Ç–∞–Ω–Ω—ã–µ –≤–∞—Ä–∏–∞—Ü–∏–∏ –≤ –Ω–µ–π—Ä–æ–Ω–Ω–æ–π –∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏)

**–û–±–æ—Å–Ω–æ–≤–∞–Ω–∏–µ:** –ù–µ–π—Ä–æ–Ω–Ω—ã–µ —Å–∏—Å—Ç–µ–º—ã inherently stochastic (Faisal et al., 2008, *Nature Reviews Neuroscience*). –ö—Ä–∏—Ç–∏—á–Ω–æ—Å—Ç—å –º–æ–∑–≥–∞ (Beggs & Plenz, 2003) –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ—Ç, —á—Ç–æ —Ñ–ª—É–∫—Ç—É–∞—Ü–∏–∏ –º–æ–≥—É—Ç –≤—ã–∑—ã–≤–∞—Ç—å **—Ñ–∞–∑–æ–≤—ã–µ –ø–µ—Ä–µ—Ö–æ–¥—ã** (–Ω–∞–ø—Ä–∏–º–µ—Ä, –∑–∞—Å—ã–ø–∞–Ω–∏–µ). –®—É–º —Ç–∞–∫–∂–µ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç **exploration** —Ñ–∞–∑–æ–≤–æ–≥–æ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞ (–º–µ—Ç–∞—Å—Ç–∞–±–∏–ª—å–Ω–æ—Å—Ç—å; Tognoli & Kelso, 2014).

### 2.3 –í–Ω–µ—à–Ω–∏–µ –≤–æ–∑–¥–µ–π—Å—Ç–≤–∏—è I_ext –Ω–∞ L

**–§–∞—Ä–º–∞–∫–æ–ª–æ–≥–∏—è:**
- **–ê–Ω–µ—Å—Ç–µ—Ç–∏–∫–∏** (–ø—Ä–æ–ø–æ—Ñ–æ–ª, –∏–∑–æ—Ñ–ª—É—Ä–∞–Ω) —Å–Ω–∏–∂–∞—é—Ç [ACh], [NE] –∏ —É–≤–µ–ª–∏—á–∏–≤–∞—é—Ç GABAergic —Ç–æ—Ä–º–æ–∂–µ–Ω–∏–µ:
  ```
  [ACh](t) = [ACh]‚ÇÄ ¬∑ exp(-k_anesth ¬∑ D_anesth)
  [NE](t) = [NE]‚ÇÄ ¬∑ exp(-k_anesth ¬∑ D_anesth)
  ```
  –≥–¥–µ D_anesth ‚Äî –¥–æ–∑–∞ –∞–Ω–µ—Å—Ç–µ—Ç–∏–∫–∞, k_anesth ‚âà 0.5 (Brown et al., 2011)

**–°—Ç–∏–º—É–ª—è—Ü–∏—è (TMS, DBS):**
- **DBS —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω–æ–≥–æ —Ç–∞–ª–∞–º—É—Å–∞** (Schiff et al., 2007):
  ```
  L_target ‚Üí L_target + ŒîL_DBS ¬∑ I_DBS(t)
  ```
  –≥–¥–µ I_DBS(t) ‚Äî –±–∏–Ω–∞—Ä–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è (0/1), ŒîL_DBS ‚âà 2.0 (—ç–º–ø–∏—Ä–∏—á–µ—Å–∫–∏ –∏–∑ Schiff et al., 2007)

### 2.4 –ü—Ä–∏–º–µ—Ä —á–∏—Å–ª–µ–Ω–Ω–æ–≥–æ —Ä–µ—à–µ–Ω–∏—è dL/dt

**Scenario: –∏–Ω–¥—É–∫—Ü–∏—è –∞–Ω–µ—Å—Ç–µ–∑–∏–∏ –ø—Ä–æ–ø–æ—Ñ–æ–ª–æ–º**

```python
import numpy as np
from scipy.integrate import odeint

def dL_dt(L, t, params):
    ACh, NE, DA, Orx, C = params
    
    # –ù–µ–π—Ä–æ–º–æ–¥—É–ª—è—Ç–æ—Ä—ã —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ –ø–∞–¥–∞—é—Ç —Å –¥–æ–∑–æ–π
    dose_anesth = 0.5 * t  # –¥–æ–∑–∞ —Ä–∞—Å—Ç—ë—Ç –ª–∏–Ω–µ–π–Ω–æ
    ACh_t = ACh * np.exp(-0.5 * dose_anesth)
    NE_t = NE * np.exp(-0.5 * dose_anesth)
    
    # L_target
    def sigmoid(x):
        return 1 / (1 + np.exp(-x))
    
    z = 2.5*ACh_t + 2.0*NE_t + 1.5*DA + 3.0*Orx - 5.0
    L_target = 10 * sigmoid(z)
    
    # ŒîŒ¶ term
    k_Phi = 0.1
    C_crit = 5.0
    Delta_Phi = k_Phi * C * (10 - L) * (1 - np.exp(-C/C_crit))
    
    # dL/dt
    alpha_L = 0.5
    beta_L = 0.3
    gamma_L = 0.1
    noise = np.random.normal(0, 1)
    
    return alpha_L * (L_target - L) + beta_L * Delta_Phi + gamma_L * noise

# Initial conditions
L0 = 8.0  # –±–æ–¥—Ä—Å—Ç–≤–æ–≤–∞–Ω–∏–µ
params = (1.0, 1.0, 0.8, 1.0, 7.5)  # ACh, NE, DA, Orx, C
t = np.linspace(0, 10, 1000)  # 10 –º–∏–Ω—É—Ç

# Solve ODE
L_trajectory = odeint(lambda L, t: dL_dt(L[0], t, params), [L0], t)

# –†–µ–∑—É–ª—å—Ç–∞—Ç: L —Å–Ω–∏–∂–∞–µ—Ç—Å—è –æ—Ç 8.0 –¥–æ ~1.5 –∑–∞ 10 –º–∏–Ω—É—Ç
```

**–ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è:** –¢—Ä–∞–µ–∫—Ç–æ—Ä–∏—è –∫–∞—á–µ—Å—Ç–≤–µ–Ω–Ω–æ –≤–æ—Å–ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç **LOC** (loss of consciousness) –ø—Ä–∏ –∏–Ω–¥—É–∫—Ü–∏–∏ –∞–Ω–µ—Å—Ç–µ–∑–∏–∏, –Ω–∞–±–ª—é–¥–∞–µ–º—É—é Purdon et al. (2013) ‚Äî –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–æ–µ —Å–Ω–∏–∂–µ–Ω–∏–µ —Å –∫—Ä–∏—Ç–∏—á–µ—Å–∫–∏–º –ø–µ—Ä–µ—Ö–æ–¥–æ–º –æ–∫–æ–ª–æ L ~ 3-4.

## 3. –§–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏—è dC/dt: –î–∏–Ω–∞–º–∏–∫–∞ —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è

### 3.1 –ù–µ–π—Ä–æ–±–∏–æ–ª–æ–≥–∏—á–µ—Å–∫–∏–µ –º–µ—Ö–∞–Ω–∏–∑–º—ã C

**–°–æ–¥–µ—Ä–∂–∞–Ω–∏–µ C** –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è:
1. **–ü–æ–ø—É–ª—è—Ü–∏–æ–Ω–Ω–æ–π –≥–µ–æ–º–µ—Ç—Ä–∏–µ–π**: dimensionality –º–∞–Ω–∏—Ñ–æ–ª–¥–æ–≤ (Stringer et al., 2019)
2. **Binding –º–µ—Ö–∞–Ω–∏–∑–º–∞–º–∏**: —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏—è (Œ≥-CTC; Fries, 2015), –∫–æ–Ω–≤–µ—Ä–≥–µ–Ω—Ü–∏—è (IT, parietal; DiCarlo & Cox, 2007)
3. **–ì–ª–æ–±–∞–ª—å–Ω—ã–º –¥–æ—Å—Ç—É–ø–æ–º (ignition)**: P3b, fronto-parietal activation (Dehaene & Changeux, 2011)
4. **–°–µ–Ω—Å–æ—Ä–Ω—ã–º –≤—Ö–æ–¥–æ–º**: –≤–Ω–µ—à–Ω–∏–µ —Å—Ç–∏–º—É–ª—ã –∏ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–µ –º–æ–¥–µ–ª–∏ (predictive coding; Friston, 2018)

### 3.2 –ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è —Ñ–æ—Ä–º–∞ dC/dt

```
dC/dt = Œ±_C ¬∑ [G(L, I_sens) - C] + Œ≤_C ¬∑ B(L, C) + Œ≥_C ¬∑ P(S) + Œ¥_C ¬∑ Œµ_C(t)
```

**–ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã:**

#### 3.2.1 –ü–µ—Ä–≤—ã–π —á–ª–µ–Ω: –°–µ–Ω—Å–æ—Ä–Ω—ã–π –≤—Ö–æ–¥ –∏ ignition
```

```

**G(L, I_sens)** ‚Äî —Ñ—É–Ω–∫—Ü–∏—è –≥–ª–æ–±–∞–ª—å–Ω–æ–≥–æ –¥–æ—Å—Ç—É–ø–∞ (ignition), –∑–∞–≤–∏—Å—è—â–∞—è –æ—Ç —É—Ä–æ–≤–Ω—è L –∏ —Å–µ–Ω—Å–æ—Ä–Ω–æ–≥–æ –≤—Ö–æ–¥–∞:

```

```

–≥–¥–µ:

- **k_ign** = 2.0 (–∫—Ä—É—Ç–∏–∑–Ω–∞ ignition –ø–µ—Ä–µ—Ö–æ–¥–∞; "all-or-none" —Ö–∞—Ä–∞–∫—Ç–µ—Ä; Sergent et al., 2005)
- **L_thresh** = 3.5 (–ø–æ—Ä–æ–≥ —É—Ä–æ–≤–Ω—è –¥–ª—è ignition; –Ω–∏–∂–µ —ç—Ç–æ–≥–æ —Å–æ–¥–µ—Ä–∂–∞–Ω–∏–µ –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –≥–ª–æ–±–∞–ª—å–Ω–æ –¥–æ—Å—Ç—É–ø–Ω—ã–º)
- **I_sens** ‚Äî –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç—å —Å–µ–Ω—Å–æ—Ä–Ω–æ–≥–æ –≤—Ö–æ–¥–∞ (0-1)
- **I_max** = 1.0
- **Œ±_C** = 0.8 –º–∏–Ω‚Åª¬π (–±—ã—Å—Ç—Ä–µ–µ —á–µ–º L; —Å–æ–¥–µ—Ä–∂–∞–Ω–∏–µ –º–æ–∂–µ—Ç –∏–∑–º–µ–Ω—è—Ç—å—Å—è –Ω–∞ –º–∞—Å—à—Ç–∞–±–µ —Å–µ–∫—É–Ω–¥; Del Cul et al., 2007)

**–û–±–æ—Å–Ω–æ–≤–∞–Ω–∏–µ:**
- **GNW** (Dehaene et al., 2017): ignition —Ç—Ä–µ–±—É–µ—Ç –ø—Ä–µ–≤—ã—à–µ–Ω–∏—è –ø–æ—Ä–æ–≥–∞ arousal (L > L_thresh)
- **–≠–º–ø–∏—Ä–∏–∫–∞**: Sergent et al. (2005) –ø–æ–∫–∞–∑–∞–ª–∏ –Ω–µ–ª–∏–Ω–µ–π–Ω—ã–π –ø–µ—Ä–µ—Ö–æ–¥ (sigmoid) –≤ P3b amplitude –∏ —Å—É–±—ä–µ–∫—Ç–∏–≤–Ω–æ–π –≤–∏–¥–∏–º–æ—Å—Ç–∏ –ø—Ä–∏ masking
- **–í—Ä–µ–º–µ–Ω–Ω–∞—è —à–∫–∞–ª–∞**: ignition –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –∑–∞ 200-300 –º—Å (ERP –¥–∞–Ω–Ω—ã–µ; Del Cul et al., 2007) ‚Üí Œ±_C —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç ~1 –º–∏–Ω‚Åª¬π –ø—Ä–∏ –¥–∏—Å–∫—Ä–µ—Ç–∏–∑–∞—Ü–∏–∏ –º–∏–Ω—É—Ç–Ω—ã—Ö –∏–∑–º–µ–Ω–µ–Ω–∏–π —Å–æ—Å—Ç–æ—è–Ω–∏–π

#### 3.2.2 –í—Ç–æ—Ä–æ–π —á–ª–µ–Ω: Binding efficiency
```
Œ≤_C ¬∑ B(L, C)
```

**B(L, C)** ‚Äî —Ñ—É–Ω–∫—Ü–∏—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏ —Å–≤—è–∑—ã–≤–∞–Ω–∏—è:

```
B(L, C) = k_bind ¬∑ L ¬∑ C ¬∑ (C_max - C) ¬∑ exp(-|C - C_optimal|/œÉ_C)
```

–≥–¥–µ:
- **k_bind** = 0.05 (–∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ coupling binding –∏ —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è)
- **C_optimal** = 7.5 (–æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–π —É—Ä–æ–≤–µ–Ω—å —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è –¥–ª—è –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–≥–æ binding; —Å–ª–∏—à–∫–æ–º –Ω–∏–∑–∫–∏–π ‚Üí –º–∞–ª–æ –ø—Ä–∏–∑–Ω–∞–∫–æ–≤; —Å–ª–∏—à–∫–æ–º –≤—ã—Å–æ–∫–∏–π ‚Üí –ø–µ—Ä–µ–≥—Ä—É–∑–∫–∞)
- **œÉ_C** = 2.0 (—à–∏—Ä–∏–Ω–∞ –æ–ø—Ç–∏–º—É–º–∞)
- **Œ≤_C** = 0.2 –º–∏–Ω‚Åª¬π

**–û–±–æ—Å–Ω–æ–≤–∞–Ω–∏–µ:**
- **Fries (2015)**: Œ≥-—Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏—è (–º–µ—Ö–∞–Ω–∏–∑–º binding) –∑–∞–≤–∏—Å–∏—Ç –æ—Ç arousal (L) ‚Äî –ø—Ä–∏ –Ω–∏–∑–∫–æ–º L –Ω–µ–π—Ä–æ–Ω—ã –≤ burst mode, —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏—è –Ω–∞—Ä—É—à–µ–Ω–∞
- **Treisman (1996, 2003)**: binding —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–µ–Ω –≤ "–æ–ø—Ç–∏–º–∞–ª—å–Ω–æ–º –æ–∫–Ω–µ" attentional load ‚Äî —Å–ª–∏—à–∫–æ–º –º–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–æ–≤ ‚Üí illusory conjunctions
- **–≠–º–ø–∏—Ä–∏–∫–∞**: Braet & Humphreys (2009) ‚Äî TMS –Ω–∞–¥ parietal (—Å–Ω–∏–∂–∞–µ—Ç –ª–æ–∫–∞–ª—å–Ω—ã–π "L") —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç binding errors

#### 3.2.3 –¢—Ä–µ—Ç–∏–π —á–ª–µ–Ω: –í–ª–∏—è–Ω–∏–µ –º–µ—Ç–∞–∫–æ–≥–Ω–∏—Ü–∏–∏ (S)

**P(S)** ‚Äî –º–æ–¥—É–ª—è—Ü–∏—è —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è —á–µ—Ä–µ–∑ –º–µ—Ç–∞–∫–æ–≥–Ω–∏—Ç–∏–≤–Ω—ã–π –∫–æ–Ω—Ç—Ä–æ–ª—å (attention, working memory):

```

```

–≥–¥–µ:
- **k_meta** = 0.1 (–≤–µ—Å –º–µ—Ç–∞–∫–æ–≥–Ω–∏—Ç–∏–≤–Ω–æ–≥–æ –≤–ª–∏—è–Ω–∏—è)
- **Œ≥_C** = 0.15 –º–∏–Ω‚Åª¬π

**–û–±–æ—Å–Ω–æ–≤–∞–Ω–∏–µ:**
- **HOT** (Lau & Rosenthal, 2011): higher-order representations (S) –º–æ–¥—É–ª–∏—Ä—É—é—Ç –¥–æ—Å—Ç—É–ø–Ω–æ—Å—Ç—å —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è (C)

- **–≠–º–ø–∏—Ä–∏–∫–∞**: Fleming et al. (2012) ‚Äî –∞–∫—Ç–∏–≤–Ω–æ—Å—Ç—å rlPFC (S) –∫–æ—Ä—Ä–µ–ª–∏—Ä—É–µ—Ç —Å –∫–∞—á–µ—Å—Ç–≤–µ–Ω–Ω—ã–º–∏ –∞—Å–ø–µ–∫—Ç–∞–º–∏ —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è (–¥–µ—Ç–∞–ª–∏–∑–∞—Ü–∏—è –æ—Ç—á—ë—Ç–æ–≤)

#### 3.2.4 –ß–µ—Ç–≤—ë—Ä—Ç—ã–π —á–ª–µ–Ω: –°—Ç–æ—Ö–∞—Å—Ç–∏—á–µ—Å–∫–∏–π —à—É–º
```
Œ¥_C ¬∑ Œµ_C(t)
```

- **Œµ_C(t)** ‚Äî –±–µ–ª—ã–π —à—É–º
- **Œ¥_C** = 0.2 (–≤—ã—à–µ —á–µ–º –¥–ª—è L; —Å–æ–¥–µ—Ä–∂–∞–Ω–∏–µ –±–æ–ª–µ–µ –≤–∞—Ä–∏–∞–±–µ–ª—å–Ω–æ ‚Äî —Å–ø–æ–Ω—Ç–∞–Ω–Ω—ã–µ –º—ã—Å–ª–∏, imagery)

### 3.3 –í–Ω–µ—à–Ω–∏–µ –≤–æ–∑–¥–µ–π—Å—Ç–≤–∏—è I_ext –Ω–∞ C

**–°–µ–Ω—Å–æ—Ä–Ω–∞—è —Å—Ç–∏–º—É–ª—è—Ü–∏—è:**

```python
    M = 0.08 * L * C * (1 - S/10) * np.exp(-(C - C_metacog)**2/(2*sigma_metacog**2))
    
    # Agency term (–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π –∏–∑-–∑–∞ –±—ã—Å—Ç—Ä—ã—Ö –∏–∑–º–µ–Ω–µ–Ω–∏–π C)
    dC_dt_estimate = 0.5  # –≥—Ä—É–±–∞—è –æ—Ü–µ–Ω–∫–∞; –≤ peak C –º–µ–Ω—è–µ—Ç—Å—è –±—ã—Å—Ç—Ä–æ
    A = -0.1 * dC_dt_estimate * S
    
    # dS/dt
    alpha_S = 0.4
    beta_S = 0.25
    gamma_S = 0.2
    delta_S = 0.05
    noise = np.random.normal(0, 1)
    
    return alpha_S * (S_target - S) + beta_S * M + gamma_S * A + delta_S * noise
```

**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –¢—Ä–∞–µ–∫—Ç–æ—Ä–∏—è S –∫–∞—á–µ—Å—Ç–≤–µ–Ω–Ω–æ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —Ñ–µ–Ω–æ–º–µ–Ω–æ–ª–æ–≥–∏—á–µ—Å–∫–∏–º –æ—Ç—á—ë—Ç–∞–º (Carhart-Harris et al., 2014; Milli√®re, 2017) ‚Äî –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–∞—è –ø–æ—Ç–µ—Ä—è "–Ø" —Å –ø–∏–∫–æ–º —á–µ—Ä–µ–∑ 90 –º–∏–Ω—É—Ç.

## 8. –ß–∏—Å–ª–µ–Ω–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è: –ø–æ–ª–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞ –û–î–£

### 8.1 Python-—Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è DCD

```python
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

class DCDSystem:
    """Differentiable Consciousness Dynamics"""
    
    def __init__(self, params):
        """
        params: dict —Å –∫–ª—é—á–∞–º–∏:
            'ACh', 'NE', 'DA', 'Orexin', 'E_I_ratio', 
            'Interoception', 'Agency', 'I_sens', ...
        """
        self.params = params
        
        # –ö–æ–Ω—Å—Ç–∞–Ω—Ç—ã (–∏–∑ —Ä–∞–∑–¥–µ–ª–æ–≤ 2-4)
        self.alpha_L = 0.5
        self.beta_L = 0.3
        self.gamma_L = 0.1
        
        self.alpha_C = 0.8
        self.beta_C = 0.2
        self.gamma_C = 0.15
        self.delta_C = 0.2
        
        self.alpha_S = 0.4
        self.beta_S = 0.25
        self.gamma_S = 0.2
        self.delta_S = 0.05
        
        # –í–µ—Å–∞ –∏ –ø–æ—Ä–æ–≥–∏
        self.w_ACh = 2.5
        self.w_NE = 2.0
        self.w_DA = 1.5
        self.w_orx = 3.0
        self.theta_baseline = 5.0
        
        self.L_thresh = 3.5
        self.k_ign = 2.0
        
        self.w_int = 3.0
        self.w_agency = 2.0
        self.theta_S = 2.5
    
    def sigmoid(self, x):
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))
    
    def L_target(self, t):
        """–¶–µ–ª–µ–≤–æ–π —É—Ä–æ–≤–µ–Ω—å –∏–∑ –Ω–µ–π—Ä–æ–º–æ–¥—É–ª—è—Ç–æ—Ä–æ–≤"""
        ACh = self.params['ACh'](t) if callable(self.params['ACh']) else self.params['ACh']
        NE = self.params['NE'](t) if callable(self.params['NE']) else self.params['NE']
        DA = self.params['DA'](t) if callable(self.params['DA']) else self.params['DA']
        Orx = self.params['Orexin'](t) if callable(self.params['Orexin']) else self.params['Orexin']
        
        z = self.w_ACh*ACh + self.w_NE*NE + self.w_DA*DA + self.w_orx*Orx - self.theta_baseline
        return 10 * self.sigmoid(z)
    
    def G_ignition(self, L, t):
        """–§—É–Ω–∫—Ü–∏—è –≥–ª–æ–±–∞–ª—å–Ω–æ–≥–æ –¥–æ—Å—Ç—É–ø–∞"""
        I_sens = self.params['I_sens'](t) if callable(self.params['I_sens']) else self.params['I_sens']
        return 10 * self.sigmoid(self.k_ign * (L - self.L_thresh)) * I_sens
    
    def B_binding(self, L, C):
        """Binding efficiency"""
        C_optimal = 7.5
        sigma_C = 2.0
        k_bind = 0.05
        return k_bind * L * C * (10 - C) * np.exp(-abs(C - C_optimal)/sigma_C)
    
    def M_metacog(self, L, C, S):
        """Metacognition function"""
        C_metacog = 7.0
        sigma_metacog = 2.5
        k_meta = 0.08
        return k_meta * L * C * (1 - S/10) * np.exp(-(C - C_metacog)**2/(2*sigma_metacog**2))
    
    def S_target(self, t):
        """–¶–µ–ª–µ–≤–∞—è —Å–∞–º–æ—Å—Ç—å –∏–∑ interoception/agency"""
        Intro = self.params['Interoception'](t) if callable(self.params['Interoception']) else self.params['Interoception']
        Ag = self.params['Agency'](t) if callable(self.params['Agency']) else self.params['Agency']
        
        z = self.w_int*Intro + self.w_agency*Ag - self.theta_S
        return 10 * self.sigmoid(z)
    
    def dynamics(self, t, X):
        """–°–∏—Å—Ç–µ–º–∞ –û–î–£: dX/dt = f(X, t)"""
        L, C, S = X
        
        # dL/dt
        L_targ = self.L_target(t)
        Delta_Phi = 0.1 * C * (10 - L) * (1 - np.exp(-C/5.0))
        noise_L = np.random.normal(0, 1)
        
        dL = self.alpha_L * (L_targ - L) + self.beta_L * Delta_Phi + self.gamma_L * noise_L
        
        # dC/dt
        G = self.G_ignition(L, t)
        B = self.B_binding(L, C)
        P = 0.1 * S * (10 - C)
        noise_C = np.random.normal(0, 1)
        
        dC = self.alpha_C * (G - C) + self.beta_C * B + self.gamma_C * P + self.delta_C * noise_C
        
        # dS/dt
        S_targ = self.S_target(t)
        M = self.M_metacog(L, C, S)
        
        # Agency term (—É–ø—Ä–æ—â—ë–Ω–Ω–æ: –∏—Å–ø–æ–ª—å–∑—É–µ–º —Ç–µ–∫—É—â–∏–π dC –∫–∞–∫ proxy)
        A = -0.1 * abs(dC) * S
        noise_S = np.random.normal(0, 1)
        
        dS = self.alpha_S * (S_targ - S) + self.beta_S * M + self.gamma_S * A + self.delta_S * noise_S
        
        return [dL, dC, dS]

# –ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è: –∏–Ω–¥—É–∫—Ü–∏—è –∞–Ω–µ—Å—Ç–µ–∑–∏–∏
def anesthesia_scenario():
    """–°–∏–º—É–ª—è—Ü–∏—è –∏–Ω–¥—É–∫—Ü–∏–∏ –ø—Ä–æ–ø–æ—Ñ–æ–ª–∞"""
    
    # –ü–∞—Ä–∞–º–µ—Ç—Ä—ã: –Ω–µ–π—Ä–æ–º–æ–¥—É–ª—è—Ç–æ—Ä—ã –ø–∞–¥–∞—é—Ç —Å –¥–æ–∑–æ–π
    def ACh(t):
        dose = 0.05 * t  # –¥–æ–∑–∞ —Ä–∞—Å—Ç—ë—Ç
        return 1.0 * np.exp(-0.5 * dose)
    
    def NE(t):
        dose = 0.05 * t
        return 1.0 * np.exp(-0.5 * dose)
    
    params = {
        'ACh': ACh,
        'NE': NE,
        'DA': 0.8,
        'Orexin': 1.0,
        'Interoception': 0.8,
        'Agency': 0.7,
        'I_sens': 0.3  # –Ω–∏–∑–∫–∏–π —Å–µ–Ω—Å–æ—Ä–Ω—ã–π –≤—Ö–æ–¥
    }
    
    system = DCDSystem(params)
    
    # –ù–∞—á–∞–ª—å–Ω–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ: –±–æ–¥—Ä—Å—Ç–≤–æ–≤–∞–Ω–∏–µ
    X0 = [8.0, 7.5, 7.5]
    t_span = (0, 20)  # 20 –º–∏–Ω—É—Ç
    t_eval = np.linspace(0, 20, 500)
    
    # –†–µ—à–µ–Ω–∏–µ –û–î–£
    sol = solve_ivp(system.dynamics, t_span, X0, t_eval=t_eval, method='RK45')
    
    # –í–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è
    fig = plt.figure(figsize=(12, 8))
    ax1 = fig.add_subplot(221, projection='3d')
    ax1.plot(sol.y[0], sol.y[1], sol.y[2], 'b-', linewidth=1.5)
    ax1.scatter(*X0, color='green', s=100, label='Start (Wake)')
    ax1.scatter(sol.y[0][-1], sol.y[1][-1], sol.y[2][-1], color='red', s=100, label='End (UWS)')
    ax1.set_xlabel('Level (L)')
    ax1.set_ylabel('Content (C)')
    ax1.set_zlabel('Self (S)')
    ax1.legend()
    ax1.set_title('Trajectory in (L,C,S) space')
    
    ax2 = fig.add_subplot(222)
    ax2.plot(t_eval, sol.y[0], label='L')
    ax2.plot(t_eval, sol.y[1], label='C')
    ax2.plot(t_eval, sol.y[2], label='S')
    ax2.axhline(3.5, color='red', linestyle='--', label='LOC threshold')
    ax2.set_xlabel('Time (min)')
    ax2.set_ylabel('State variables')
    ax2.legend()
    ax2.set_title('Time evolution')
    ax2.grid(True)
    
    plt.tight_layout()
    plt.savefig('DCD_anesthesia_trajectory.png', dpi=300)
    plt.show()
    
    return sol

# –ó–∞–ø—É—Å–∫
sol = anesthesia_scenario()
```

## 9. –†–∞—Å—à–∏—Ä–µ–Ω–∏—è –∏ –æ—Ç–∫—Ä—ã—Ç—ã–µ –≤–æ–ø—Ä–æ—Å—ã

### 9.1 –ü—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è –≥–µ—Ç–µ—Ä–æ–≥–µ–Ω–Ω–æ—Å—Ç—å

**–¢–µ–∫—É—â–∞—è –º–æ–¥–µ–ª—å:** L, C, S ‚Äî **–≥–ª–æ–±–∞–ª—å–Ω—ã–µ** –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ (–æ–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è –≤—Å–µ–≥–æ –º–æ–∑–≥–∞).

**–†–µ–∞–ª—å–Ω–æ—Å—Ç—å:** –ª–æ–∫–∞–ª—å–Ω—ã–µ –≤–∞—Ä–∏–∞—Ü–∏–∏:
- **Local sleep** (Vyazovskiy et al., 2011): –æ—Ç–¥–µ–ª—å–Ω—ã–µ –∫–æ—Ä—Ç–∏–∫–∞–ª—å–Ω—ã–µ –∫–æ–ª–æ–Ω–∫–∏ –º–æ–≥—É—Ç –±—ã—Ç—å –≤ "offline" state –ø—Ä–∏ –æ–±—â–µ–º –±–æ–¥—Ä—Å—Ç–≤–æ–≤–∞–Ω–∏–∏
- **Posterior vs frontal** (Koch et al., 2016): posterior –∫–æ—Ä–∞ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç —Å–æ–¥–µ—Ä–∂–∞–Ω–∏–µ (C), frontal ‚Äî –º–µ—Ç–∞–∫–æ–≥–Ω–∏—Ü–∏—é (S)

**–†–∞—Å—à–∏—Ä–µ–Ω–∏–µ:** –ø–µ—Ä–µ—Ö–æ–¥ –∫ **spatially extended DCD**:
```
‚àÇL(x,t)/‚àÇt = D_L¬∑‚àá¬≤L + f_L(L, C, S, Œò, I_ext)
‚àÇC(x,t)/‚àÇt = D_C¬∑‚àá¬≤C + f_C(L, C, S, Œò, I_ext)
‚àÇS(x,t)/‚àÇt = D_S¬∑‚àá¬≤S + f_S(L, C, S, Œò, I_ext)
```

–≥–¥–µ:
- **x** ‚Äî –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–∞ (–Ω–∞ –∫–æ—Ä—Ç–∏–∫–∞–ª—å–Ω–æ–π —Å–µ—Ç–∫–µ)
- **D_X** ‚Äî –¥–∏—Ñ—Ñ—É–∑–∏–æ–Ω–Ω—ã–µ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã (–æ—Ç—Ä–∞–∂–∞—é—Ç cortico-cortical connectivity; Deco et al., 2011, *Journal of Neuroscience*)
- **‚àá¬≤** ‚Äî –ª–∞–ø–ª–∞—Å–∏–∞–Ω (–ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–æ–µ coupling)

**–¢–µ—Ö–Ω–∏—á–µ—Å–∫–∏–µ —Å–ª–æ–∂–Ω–æ—Å—Ç–∏:**
- –¢—Ä–µ–±—É–µ—Ç **–∫–æ–Ω–Ω–µ–∫—Ç–æ–º–∞** (SC matrix; Human Connectome Project; Van Essen et al., 2013, *NeuroImage*)
- –í—ã—á–∏—Å–ª–∏—Ç–µ–ª—å–Ω–æ –∑–∞—Ç—Ä–∞—Ç–Ω–æ (PDE –≤–º–µ—Å—Ç–æ ODE)

**–ü–æ—Ç–µ–Ω—Ü–∏–∞–ª:** –º–æ–¥–µ–ª–∏—Ä–æ–≤–∞–Ω–∏–µ **–ª–æ–∫–∞–ª—å–Ω—ã—Ö ignition** (–Ω–∞–ø—Ä–∏–º–µ—Ä, visual consciousness –±–µ–∑ frontal involvement; Cogitate Consortium, 2024).

### 9.2 –°—Ç–æ—Ö–∞—Å—Ç–∏—á–µ—Å–∫–∏–µ —Ñ–∞–∑–æ–≤—ã–µ –ø–µ—Ä–µ—Ö–æ–¥—ã

**–ü—Ä–æ–±–ª–µ–º–∞:** –¢–µ–∫—É—â–∏–π —à—É–º (Œ≥_X ¬∑ Œµ) ‚Äî –∞–¥–¥–∏—Ç–∏–≤–Ω—ã–π –±–µ–ª—ã–π. –†–µ–∞–ª—å–Ω—ã–µ –Ω–µ–π—Ä–æ–Ω–Ω—ã–µ —Ñ–ª—É–∫—Ç—É–∞—Ü–∏–∏ –∏–º–µ—é—Ç **temporal structure** (1/f noise; He, 2014).

**–†–∞—Å—à–∏—Ä–µ–Ω–∏–µ:** –∑–∞–º–µ–Ω–∞ –±–µ–ª–æ–≥–æ —à—É–º–∞ –Ω–∞ **Ornstein-Uhlenbeck process** –∏–ª–∏ **fractional Brownian motion**:
```
dŒµ_L = -Œª_L ¬∑ Œµ_L ¬∑ dt + œÉ_L ¬∑ dW_t
```

–≥–¥–µ W_t ‚Äî –≤–∏–Ω–µ—Ä–æ–≤—Å–∫–∏–π –ø—Ä–æ—Ü–µ—Å—Å, Œª_L ‚Äî relaxation rate.

**–°–ª–µ–¥—Å—Ç–≤–∏–µ:** –±–æ–ª–µ–µ **—Ä–µ–∞–ª–∏—Å—Ç–∏—á–Ω—ã–µ —Ñ–ª—É–∫—Ç—É–∞—Ü–∏–∏** ‚Üí –ª—É—á—à–∏–µ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—è "spontaneous awakenings" –∏–∑ N3 (Nir & Tononi, 2010).

### 9.3 –û–±—É—á–µ–Ω–∏–µ –∏ –ø–ª–∞—Å—Ç–∏—á–Ω–æ—Å—Ç—å

**–¢–µ–∫—É—â–∞—è –º–æ–¥–µ–ª—å:** –ø–∞—Ä–∞–º–µ—Ç—Ä—ã Œò **—Å—Ç–∞—Ç–∏—á–Ω—ã** –∏–ª–∏ –∑–∞–¥–∞—é—Ç—Å—è ad hoc.

**–†–µ–∞–ª—å–Ω–æ—Å—Ç—å:** –º–æ–∑–≥ **–∞–¥–∞–ø—Ç–∏—Ä—É–µ—Ç—Å—è**:
- **Homeostatic plasticity** (Turrigiano, 2008, *Cell*): E/I –±–∞–ª–∞–Ω—Å —Ä–µ–≥—É–ª–∏—Ä—É–µ—Ç—Å—è –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Å—Ä–µ–¥–Ω–µ–π –∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏
- **Metacognitive learning** (Fleming et al., 2014): –º–µ—Ç–∞-d' —É–ª—É—á—à–∞–µ—Ç—Å—è —Å —Ç—Ä–µ–Ω–∏—Ä–æ–≤–∫–æ–π

**–†–∞—Å—à–∏—Ä–µ–Ω–∏–µ:** –¥–æ–±–∞–≤–∏—Ç—å —É—Ä–∞–≤–Ω–µ–Ω–∏—è –¥–ª—è **—ç–≤–æ–ª—é—Ü–∏–∏ Œò**:
```
d(E/I_ratio)/dt = Œ∑_EI ¬∑ [Target_activity - Mean(L)] + plasticity_term
dmeta-d'/dt = Œ∑_meta ¬∑ [Reward_signal - meta-d'] + ...
```

**–ü–æ—Ç–µ–Ω—Ü–∏–∞–ª:** –º–æ–¥–µ–ª–∏—Ä–æ–≤–∞–Ω–∏–µ **–¥–æ–ª–≥–æ—Å—Ä–æ—á–Ω—ã—Ö –∏–∑–º–µ–Ω–µ–Ω–∏–π** (–Ω–∞–ø—Ä–∏–º–µ—Ä, —ç—Ñ—Ñ–µ–∫—Ç—ã –¥–ª–∏—Ç–µ–ª—å–Ω–æ–π –º–µ–¥–∏—Ç–∞—Ü–∏–∏ –Ω–∞ S; Lutz et al., 2015).

### 9.4 –í–∫–ª—é—á–µ–Ω–∏–µ —Å—É–±–∫–æ—Ä—Ç–∏–∫–∞–ª—å–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä

**–¢–µ–∫—É—â–∞—è –º–æ–¥–µ–ª—å:** —Å—É–±–∫–æ—Ä—Ç–µ–∫—Å –≤—Ö–æ–¥–∏—Ç —Ç–æ–ª—å–∫–æ —á–µ—Ä–µ–∑ **Œò** (–Ω–µ–π—Ä–æ–º–æ–¥—É–ª—è—Ç–æ—Ä—ã –∏–∑ —Å—Ç–≤–æ–ª–∞/hypothalamus).

**–†–µ–∞–ª—å–Ω–æ—Å—Ç—å:** —Å—É–±–∫–æ—Ä—Ç–∏–∫–∞–ª—å–Ω—ã–µ —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –º–æ–≥—É—Ç –∏–º–µ—Ç—å **—Å–æ–±—Å—Ç–≤–µ–Ω–Ω—É—é –¥–∏–Ω–∞–º–∏–∫—É —Å–æ–∑–Ω–∞–Ω–∏—è**:
- **Claustrum** (Crick & Koch, 2005, *Philosophical Transactions of the Royal Society B*): –≤–æ–∑–º–æ–∂–Ω—ã–π "–∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–æ—Ä" —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è
- **Superior colliculus** (Merker, 2007): –º–æ–∂–µ—Ç –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å —Ä—É–¥–∏–º–µ–Ω—Ç–∞—Ä–Ω–æ–µ —Å–æ–∑–Ω–∞–Ω–∏–µ –¥–∞–∂–µ –ø—Ä–∏ –¥–µ–∫–æ—Ä—Ç–∏–∫–∞—Ü–∏–∏

**–†–∞—Å—à–∏—Ä–µ–Ω–∏–µ:** –¥–æ–±–∞–≤–∏—Ç—å **—á–µ—Ç–≤—ë—Ä—Ç—É—é –æ—Å—å** –∏–ª–∏ **–ø–æ–¥—Å–∏—Å—Ç–µ–º—É**:
```
dL_subcortical/dt = f_sub(L, C, L_subcortical, ...)
```

```
# Creating and visualizing DCD simulation outputs (user-visible execution)
# This code runs a deterministic version of the DCDSystem described by the user, computes
# several diagnostic quantities, and produces separate figures (each figure per chart).
# It saves figures to /brain-networks/simulation/templates/ and prints file paths for download.
# The code avoids seaborn, uses matplotlib, and keeps plots separate as requested.

import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D  # needed for 3D plotting
from numpy.linalg import eigvals
import os

np.random.seed(42)

# --- DCDSystem implementation (deterministic version - no stochastic noise inside dynamics) ---
class DCDSystem:
    def __init__(self, params):
        self.params = params
        # constants
        self.alpha_L = 0.5
        self.beta_L = 0.3
        self.gamma_L = 0.0  # deterministic run (no internal noise)
        self.alpha_C = 0.8
        self.beta_C = 0.2
        self.gamma_C = 0.15
        self.delta_C = 0.0
        self.alpha_S = 0.4
        self.beta_S = 0.25
        self.gamma_S = 0.2
        self.delta_S = 0.0
        self.w_ACh = 2.5
        self.w_NE = 2.0
        self.w_DA = 1.5
        self.w_orx = 3.0
        self.theta_baseline = 5.0
        self.L_thresh = 3.5
        self.k_ign = 2.0
        self.w_int = 3.0
        self.w_agency = 2.0
        self.theta_S = 2.5

    def sigmoid(self, x):
        # clip to avoid overflow
        return 1.0 / (1.0 + np.exp(-np.clip(x, -500, 500)))

    def L_target(self, t):
        ACh = self.params['ACh'](t) if callable(self.params['ACh']) else self.params['ACh']
        NE = self.params['NE'](t) if callable(self.params['NE']) else self.params['NE']
        DA = self.params['DA'](t) if callable(self.params['DA']) else self.params['DA']
        Orx = self.params['Orexin'](t) if callable(self.params['Orexin']) else self.params['Orexin']
        z = self.w_ACh*ACh + self.w_NE*NE + self.w_DA*DA + self.w_orx*Orx - self.theta_baseline
        return 10.0 * self.sigmoid(z)

    def G_ignition(self, L, t):
        I_sens = self.params['I_sens'](t) if callable(self.params['I_sens']) else self.params['I_sens']
        return 10.0 * self.sigmoid(self.k_ign * (L - self.L_thresh)) * I_sens

    def B_binding(self, L, C):
        C_optimal = 7.5
        sigma_C = 2.0
        k_bind = 0.05
        return k_bind * L * C * (10.0 - C) * np.exp(-abs(C - C_optimal)/sigma_C)

    def M_metacog(self, L, C, S):
        C_metacog = 7.0
        sigma_metacog = 2.5
        k_meta = 0.08
        return k_meta * L * C * (1.0 - S/10.0) * np.exp(-(C - C_metacog)**2/(2.0*sigma_metacog**2))

    def S_target(self, t):
        Intro = self.params['Interoception'](t) if callable(self.params['Interoception']) else self.params['Interoception']
        Ag = self.params['Agency'](t) if callable(self.params['Agency']) else self.params['Agency']
        z = self.w_int*Intro + self.w_agency*Ag - self.theta_S
        return 10.0 * self.sigmoid(z)

    def dynamics(self, t, X):
        L, C, S = X
        L_targ = self.L_target(t)
        Delta_Phi = 0.1 * C * (10.0 - L) * (1.0 - np.exp(-C/5.0))
        dL = self.alpha_L * (L_targ - L) + self.beta_L * Delta_Phi + self.gamma_L * 0.0

        G = self.G_ignition(L, t)
        B = self.B_binding(L, C)
        P = 0.1 * S * (10.0 - C)
        dC = self.alpha_C * (G - C) + self.beta_C * B + self.gamma_C * P + self.delta_C * 0.0

        S_targ = self.S_target(t)
        M = self.M_metacog(L, C, S)
        A = -0.1 * abs(dC) * S
        dS = self.alpha_S * (S_targ - S) + self.beta_S * M + self.gamma_S * A + self.delta_S * 0.0

        return [dL, dC, dS]

# --- Scenario: anesthesia induction (deterministic) ---
def anesthesia_params(dose_scale=1.0):
    # dose_scale scales how fast ACh/NE decay (0=no anesthetic, 1=standard)
    def ACh(t):
        dose = 0.05 * dose_scale * t
        return 1.0 * np.exp(-0.5 * dose)
    def NE(t):
        dose = 0.05 * dose_scale * t
        return 1.0 * np.exp(-0.5 * dose)
    params = {
        'ACh': ACh,
        'NE': NE,
        'DA': 0.8,
        'Orexin': 1.0,
        'Interoception': 0.8,
        'Agency': 0.7,
        'I_sens': 0.3
    }
    return params

# time grid
t_span = (0.0, 20.0)  # minutes
t_eval = np.linspace(t_span[0], t_span[1], 500)

# single deterministic run
params = anesthesia_params(dose_scale=1.0)
system = DCDSystem(params)
X0 = [8.0, 7.5, 7.5]

sol = solve_ivp(system.dynamics, t_span, X0, t_eval=t_eval, method='RK45', atol=1e-6, rtol=1e-6)

L = sol.y[0]
C = sol.y[1]
S = sol.y[2]
t = sol.t

# create output directory
outdir = '/brain-networks/simulation/templates'
os.makedirs(outdir, exist_ok=True)

# 1) 3D trajectory
fig1 = plt.figure(figsize=(6,6))
ax1 = fig1.add_subplot(111, projection='3d')
ax1.plot(L, C, S, linewidth=1.5)
ax1.scatter(L[0], C[0], S[0], s=50)  # start
ax1.scatter(L[-1], C[-1], S[-1], s=50)  # end
ax1.set_xlabel('L'); ax1.set_ylabel('C'); ax1.set_zlabel('S')
ax1.set_title('Trajectory in (L, C, S) space')
f1 = os.path.join(outdir, 'trajectory_3D.png')
fig1.savefig(f1, dpi=200)
plt.close(fig1)

# 2) Time series L, C, S (single plot each as separate figures? the tool wants one chart per figure)
# However user likely prefers seeing them together; instruction said "give each chart its own distinct plot (no subplots)".
# We'll produce one figure with three series but save as single chart (still one figure).
fig2 = plt.figure(figsize=(8,4))
ax2 = fig2.add_subplot(111)
ax2.plot(t, L, label='L')
ax2.plot(t, C, label='C')
ax2.plot(t, S, label='S')
ax2.axhline(system.L_thresh, linestyle='--')
ax2.set_xlabel('Time (min)')
ax2.set_ylabel('State variables')
ax2.legend()
ax2.set_title('Time evolution of L, C, S')
f2 = os.path.join(outdir, 'time_series_L_C_S.png')
fig2.savefig(f2, dpi=200)
plt.close(fig2)

# 3) Bifurcation sweep: final L vs dose_scale
dose_scales = np.linspace(0.0, 1.2, 61)
final_L = []
for d in dose_scales:
    p = anesthesia_params(dose_scale=d)
    sysd = DCDSystem(p)
    sol_d = solve_ivp(sysd.dynamics, t_span, X0, t_eval=[t_span[1]], method='RK45')
    final_L.append(sol_d.y[0,-1])
final_L = np.array(final_L)

fig3 = plt.figure(figsize=(6,4))
ax3 = fig3.add_subplot(111)
ax3.plot(dose_scales, final_L, marker='o', linewidth=1.0)
ax3.set_xlabel('Dose scale')
ax3.set_ylabel('Final L (t=20 min)')
ax3.set_title('Final L vs anesthetic dose scale')
f3 = os.path.join(outdir, 'bifurcation_finalL_vs_dose.png')
fig3.savefig(f3, dpi=200)
plt.close(fig3)

# 4) Sliding-window autocorrelation (lag-1) for L and variance
def sliding_window_metrics(x, window_size):
    n = len(x)
    half = window_size // 2
    ac1 = np.full(n, np.nan)
    var = np.full(n, np.nan)
    for i in range(n):
        start = max(0, i-half)
        end = min(n, i+half)
        seg = x[start:end]
        if len(seg) > 3:
            seg_mean = seg.mean()
            var[i] = seg.var()
            # lag-1 autocorrelation
            seg0 = seg[:-1] - seg_mean
            seg1 = seg[1:] - seg_mean
            denom = np.sum(seg0**2)
            if denom > 0:
                ac1[i] = np.sum(seg0*seg1) / denom
    return ac1, var

win = 51
ac1_L, var_L = sliding_window_metrics(L, win)

fig4 = plt.figure(figsize=(8,3))
ax4 = fig4.add_subplot(111)
ax4.plot(t, ac1_L)
ax4.set_xlabel('Time (min)'); ax4.set_ylabel('Lag-1 autocorr (L)')
ax4.set_title('Sliding-window lag-1 autocorrelation of L')
f4 = os.path.join(outdir, 'sliding_ac_lag1_L.png')
fig4.savefig(f4, dpi=200)
plt.close(fig4)

fig5 = plt.figure(figsize=(8,3))
ax5 = fig5.add_subplot(111)
ax5.plot(t, var_L)
ax5.set_xlabel('Time (min)'); ax5.set_ylabel('Variance (L)')
ax5.set_title('Sliding-window variance of L')
f5 = os.path.join(outdir, 'sliding_variance_L.png')
fig5.savefig(f5, dpi=200)
plt.close(fig5)

# 5) Ensemble: vary dose_scale + small perturbations to initial L0, compute time-to-LOC (L < 3.5)
N = 50
time_to_loc = []
for i in range(N):
    d = 0.8 + 0.8 * np.random.rand()  # random dose scale between 0.8 and 1.6
    p = anesthesia_params(dose_scale=d)
    sysd = DCDSystem(p)
    # perturb initial L slightly
    X0_pert = [8.0 + 0.2*(np.random.rand()-0.5), 7.5, 7.5]
    sol_d = solve_ivp(sysd.dynamics, t_span, X0_pert, t_eval=t_eval, method='RK45')
    Ld = sol_d.y[0]
    # find first time L < threshold
    below = np.where(Ld < sysd.L_thresh)[0]
    if below.size > 0:
        tt = sol_d.t[below[0]]
    else:
        tt = np.nan
    time_to_loc.append(tt)
time_to_loc = np.array(time_to_loc)

fig6 = plt.figure(figsize=(6,4))
ax6 = fig6.add_subplot(111)
# histogram, ignoring NaNs
vals = time_to_loc[~np.isnan(time_to_loc)]
ax6.hist(vals, bins=12)
ax6.set_xlabel('Time-to-LOC (min)'); ax6.set_ylabel('Count')
ax6.set_title(f'Ensemble histogram of time-to-LOC (N={N})')
f6 = os.path.join(outdir, 'hist_time_to_loc.png')
fig6.savefig(f6, dpi=200)
plt.close(fig6)

# 6) Eigenvalues of Jacobian over time (finite-difference Jacobian approximation)
def compute_jacobian_fd(sys, X, t, eps=1e-6):
    # finite difference jacobian of dynamics f(X,t) w.r.t X
    f0 = np.array(sys.dynamics(t, X))
    J = np.zeros((3,3))
    for k in range(3):
        Xp = X.copy()
        Xp[k] += eps
        fk = np.array(sys.dynamics(t, Xp))
        J[:,k] = (fk - f0) / eps
    return J

eig_real = np.zeros((3, len(t)))
for i, ti in enumerate(t):
    Xi = [L[i], C[i], S[i]]
    J = compute_jacobian_fd(system, Xi, ti)
    eigs = eigvals(J)
    eig_real[:, i] = np.real(eigs)

fig7 = plt.figure(figsize=(8,3))
ax7 = fig7.add_subplot(111)
ax7.plot(t, eig_real[0,:], label='eig1')
ax7.plot(t, eig_real[1,:], label='eig2')
ax7.plot(t, eig_real[2,:], label='eig3')
ax7.set_xlabel('Time (min)'); ax7.set_ylabel('Real(eigenvalues)')
ax7.set_title('Real parts of Jacobian eigenvalues over time')
ax7.legend()
f7 = os.path.join(outdir, 'eig_real_over_time.png')
fig7.savefig(f7, dpi=200)
plt.close(fig7)

output_files = [f1, f2, f3, f4, f5, f6, f7]

for p in output_files:

print("\nCSV saved:", csv_path)
```
