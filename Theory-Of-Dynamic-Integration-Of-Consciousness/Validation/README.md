2. **–ü–æ–ª–æ—Å–æ–≤–∞—è —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏—è (1‚Äì100 –ì—Ü)**

   * **Low-pass (80‚Äì100 –ì—Ü)** –¥–ª—è –ø–æ–¥–∞–≤–ª–µ–Ω–∏—è –º—ã—à–µ—á–Ω–æ–π –∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏ –∏ –≤–Ω–µ—à–Ω–µ–≥–æ –≤—ã—Å–æ–∫–æ—á–∞—Å—Ç–æ—Ç–Ω–æ–≥–æ —à—É–º–∞.
